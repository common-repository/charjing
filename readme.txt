This Plugin allows you to  manage products and receive payments using authorize.net merchant account.

== Description ==
Charjing provides a very simple products billing and subscription management.using custom post type

== Installation ==

1. Upload the entire `Charjing` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. go to Setings->Charjing Options page, and enter your authorize.net account api details

You will find 'Charjing Products' menu in your WordPress admin panel.
where you can create products list, set setup price,recurring price and description of your products
To view products list go to  http://yoursite.com/chg-products/
choose a product, Click buy now and checkout through authorize.net
